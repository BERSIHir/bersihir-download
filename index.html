How to install

Android · APK

Sideload on Android

  1. Tap Download APK on this page from your Android device.
  2. When prompted, allow Install unknown apps for your browser.
  3. Open the downloaded file and confirm Install.
  4. If you have an older version, uninstall it first to avoid a signature mismatch.
iOS · IPA

Install over the air

  1. Open this page in Safari on the iPhone you'll use to test.
  2. Confirm your UDID has been added to our provisioning profile — message the team if unsure.
  3. Tap Install over the air and accept the system prompt.
  4. In Settings → General → VPN & Device Management, trust the BERSIHir profile.
